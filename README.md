Sendgrid-Statsd Docker Container
================================

This is the source for **jannkleen/sendgrid-statsd** in the docker index.

Installation
------------

- The easy way:

    ```
    docker pull jannkleen/sendgrid-statsd
    ```

- The other way:

    ```
    git clone https://github.com/JannKleen/sendgrid-statsd.git

    cd sendgrid-statsd

    docker build .
    ```
